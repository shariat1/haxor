## Installation :
 
* `apt update` 
* `apt upgrade` 
* `apt install git -y` 
* `git clone https://github.com/darknethaxor/haxor.git` 
* `cd haxor` 
* `chmod +x setup.sh` 
* `bash setup.sh` 

## For Run : 
* `haxor`
